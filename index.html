INTERFAZ DE CONTROL CON ASISTENTE DE VOZ

Este es un proyecto enfocado en Dómotica (IoT) - Con este proyecto se buscó el control de varios sensores y actuadores (Hardware) a través de comandos de voz totalmente personalizados gracias a un framework de JavaScript llamado Artyom.js, el cuál nos permite manipular las funcionalidades de nuestra pagina web y accionarla por medio de los comandos en la interfaz gráfica.

Ademas de esta libreria, la interconexión se realiza gracias al Arduino IDE vinculando los sistemas fisicos por medio de una placa de desarrollo llamada ESP8266 para posteriormente; usando la estructura de datos 'arboles' en Firebase (Base de datos en tiempo real), poder conectar los sistemas y con ello, manipularlos desde la web.

JUEGO DINOCHROME AUTOMATIZADO
CON ALGORITMOS GENETICOS

En este proyecto se realizo la automatizacion del famoso juego 'DinoChrome' utilizando y aplicando la teoría de los Algoritmos Genéticos tanto para la optmización como para la automatización de resultados.
Para esto, desde el codigo libre y orginal del juego se creo una clase en JavaScript para utilizar 10 dinosaurios (población inicial) por cada generacion (partida) y poderle aplicar diversos operadores genéticos (generación de cromosomas y alelos, selección, cruce, mutación) esto con el fin de encontrar a los 2 mejores individuos por cada generación y con ello optimizar el puntaje obtenido hasta encontrar a la mejor generación de Dinosuarios, todo esto de forma automática.
